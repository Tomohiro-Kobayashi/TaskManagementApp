# 仕事で使う用のタスク管理アプリ

## 動作環境  
Mac OS Monterey

### npmで管理しているバージョン

- react@17.0.2
- react-dom@17.0.2
- react-beautiful-dnd@13.1.0
- react-scripts@5.0.0


## 目標
Docker-composeで立ち上げて、ローカルで使用するタスク管理アプリを作成する。

### フロント
React 

### バックエンド
Go

### データベース
mongoDB 

## 背景
会社のセキュリティ上、学生の頃に使用していたTrelloを使用することができない。また、類似したサービスも使用できないことから、自分で自分が使いやすいタスク管理アプリを作成することにした。
